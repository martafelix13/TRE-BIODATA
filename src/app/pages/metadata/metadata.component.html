Metadata Handling

Metadata is data that describes other data. It summarizes basic information, making it easier to find and use specific data instances. For example, an image may include metadata describing the picture's size, color depth, resolution, creation time, and more. Similarly, a text document's metadata may include information about the author, document length, creation date, and summary.

Metadata helps understand, use, and manage data by describing its content, quality, condition, origin, and more. It's essential for efficient data management and searchability.

Metadata Submission

To submit metadata to the system database, follow the steps below:

Step 1: Login into the System

Access the system using your federated credentials.

Step 2: Access the Metadata Submission Form

Click on the "Metadata" tab to access the submission form.

Step 3: Fill in the Metadata Submission Form

Provide the required information in the metadata submission form.

Step 4: Submit the Metadata

Click the "Submit" button to finalize the submission.
