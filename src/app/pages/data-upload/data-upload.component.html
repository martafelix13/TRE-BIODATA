About Crypt4GH

Crypt4GH is a secure file format and encryption tool designed for genomic data sharing, developed by the Global Alliance for Genomics and Health (GA4GH). It enables you to encrypt genomic files so they can be safely transferred and stored, ensuring only authorized users can decrypt and access the data.

For more details, see the official Crypt4GH repository.

How to Install Crypt4GH

  1. Install via pip (recommended): 
    pip install crypt4gh
  2. Or install from source: 
    
            git clone https://github.com/EGA-archive/crypt4gh.git
            cd crypt4gh
            pip install .
  3. Verify installation: 
    crypt4gh --help

Crypt4GH requires Python 3.6 or higher. For more advanced usage and troubleshooting, refer to the GitHub documentation.
